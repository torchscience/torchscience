# torchscience

PyTorch operators for mathematics, science, and engineering.

## Domains

- Geometry
- Graph algorithms
- Initial value problems
- Integral transforms
- Linear algebra
- Optimization
  - Root-finding
- Probability
- Special functions
- Statistics
