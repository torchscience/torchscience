Installation
============

Requirements
------------

* Python >= 3.13
* PyTorch >= 2.9.1
* NumPy >= 2.3.5

Install from PyPI
-----------------

.. code-block:: bash

   pip install torchscience

Install from source
-------------------

.. code-block:: bash

   git clone https://github.com/0x00b1/torchscience.git
   cd torchscience
   pip install .
