# torchscience

PyTorch operators for mathematics, science, and engineering.

## Domains

- Geometry
- Graphs
- Initial value problems
- Integral transforms
- Integration
- Linear algebra
- Optimization
- Probability
- Root-finding
- Special functions
- Statistics
